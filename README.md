# PG29ViniciusKen_Mobile_Final - Subway Cutters

Subway cutters is a 2D mobile game where the player has to procedurally slice fruits that are being thrown into the air, while avoiding the bombs. 
Made with Unity 6000.3.12f1 for Android.

## Features

- Procedural Mesh cutting for fruits and bombs
- Swipe Input with Trail VFX
- Unity ADs
